Installer [Zurb Foundation](http://foundation.zurb.com/docs/sass.html) :

    $ [sudo] gem install compass
    $ [sudo] gem install zurb-foundation

Lancer le serveur (sur le port 4000) :

    $ jekyll --server --auto

pour voir le résultat : http://localhost:4000
