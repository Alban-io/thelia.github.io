public string asseticize(string $asset_path, string $output_path, string $output_url, string $asset_type, array $filters, boolean $debug, boolean $dev_mode = false)

Generates assets from $assetpath in $outputpath, using $filters.

Parameters

string $asset_path the full path to the asset file (or file collection)
string $output_path the full disk path to the output directory (shoud be visible to web server)
string $output_url the URL to the generated asset directory
string $asset_type the asset type: css, js, ... The generated files will have this extension. Pass an empty string to use the asset source extension.
array $filters a list of filters, as defined below (see switch($filter_name) ...)
boolean $debug true / false
boolean $dev_mode true / false. If true, assets are not cached and always compiled.

Return Value

string The URL to the generated asset file.

Exceptions

InvalidArgumentException if an invalid filter name is found
