public prepareAssets(string $sourceAssetsDirectory, string $webAssetsDirectoryBase, $webAssetsTemplate $webAssetsTemplate, string $webAssetsKey)

Prepare an asset directory by checking that no changes occured in the source directory.

If any change is detected, the whole asset directory is copied in the web space.

Parameters

string $sourceAssetsDirectory the full path to the source asstes directory
string $webAssetsDirectoryBase the base directory of the web based asset directory
$webAssetsTemplate $webAssetsTemplate
string $webAssetsKey the assets key : module name or 0 for base template

Exceptions

RuntimeException if something goes wrong.
